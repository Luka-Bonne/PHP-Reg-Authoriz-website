# PHP-Reg-Authoriz-website

Стек:
- HTML5
- CSS3
- PHP (серверная часть) с использованием MAMP
- БД

Реализованы страницы:
- Главная
- Инфо
- Профиль (страница видна только для авторизованных пользователей)
- Регистрация
- Авторизация с использованием reCAPTCHA
 (Кнопки для перехода на страницы авторизации и регистрации не видны авторизованному пользователю, их заменяет кнопка выхода из профиля)
